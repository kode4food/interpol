Creates a Null Writer. All calls to this writer find their way into the bit bucket. Its primary purpose is to support the background rendering of modules in order to yield their exported symbols.

function createNullWriter() { return freezeObject({ startRender: noOp, endRender: noOp, startElement: noOp, selfCloseElement: noOp, endElement: noOp, comment: noOp, docType: noOp, content: noOp, rawContent: noOp }); }

Exported Functions

exports.createNullWriter = createNullWriter; interpol.createNullWriter = createNullWriter;
